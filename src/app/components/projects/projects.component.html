  • Fucion de lectura de codigos desde la camara

Social Duoc

Una plataforma web que busca fomentar la comunicación entre integrantes de una institución mediante:

  • Uso de Dajango, Jquery, datatable, Ajax, entre otros
  • Notificaciones usando Firebase
  • Chat (privado y grupal) en tiempo real
  • Administración de chat grupal
  • Administración con gestor de contenido
  • Personalización de perfil

RegistroApp

  • Perfil de Creador de eventos(docente) y participantes (alumnos)
  • Notificaciones usando Firebase
  • Lectura y generación de QR
  • Historial de asistencias
  • Personalización de perfil
